Bulletin Admin

Enter your access code to edit and publish the weekly bulletin.

Setting up a new editor?
  1. Add their GitHub account as a collaborator on ccav-01/bulletin — repo Settings → Collaborators, role: Write.
  2. Go to GitHub → Settings → Personal access tokens (fine-grained) while signed in as them, and create a token scoped to ccav-01/bulletin with Contents: Read & Write.
  3. Send them the token as their access code. They paste it above once — the browser saves it from there.
  4. To remove access, revoke the token in GitHub.
